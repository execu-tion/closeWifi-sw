# 项目目录结构简介

本仓库是一个嵌入式开发工具集，目录结构如下：

## 📁 **dts/** - 设备树生成
- **作用**：生成和配置Linux内核启动所需的设备树文件
- 包含设备树源文件(.dts)和编译工具
- 用于定义硬件外设和系统配置

## ⚙️ **fsbl/** - 第一级引导程序
- **作用**：硬件初始化的最底层引导程序
- 负责初始化DDR、时钟等基础硬件
- 为U-Boot和操作系统启动做准备

## 📜 **scripts/** - 实用脚本
- **作用**：自动化构建、测试和部署流程
- 包含构建脚本、部署脚本、工具脚本
- 简化开发流程，提高效率

## 🐧 **uboot/** - 引导加载程序
- **作用**：主引导加载程序，支持多种启动方式
- 基于Xilinx官方的u-boot-xlnx仓库
- 支持网络启动、SD卡启动、设备树等功能

---

**总结**：从硬件初始化(fsbl) → 引导加载(uboot) → 系统配置(dts)，配合自动化脚本(scripts)

**注意**：请手动应用补丁
```
git submodule update --init
chmod +x scripts/apply-patches.sh
./scripts/apply-patches.sh
```
